# Outputs Directory

This directory contains all extraction outputs.

## Structure

```
outputs/
├── extractions/        # Excel files with extracted data
├── reports/            # Evaluation and processing reports
└── database/           # SQLite database files
```

## Output Files

- **Excel Files**: Named with timestamp, e.g., `invoice_extractions_20260121_143022.xlsx`
- **Database**: `invoice_extractions.db` (SQLite)
- **Reports**: JSON evaluation reports with accuracy metrics
